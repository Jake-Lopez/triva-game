# Trivia Game - General Knowledge

A fun and interactive web-based trivia game that tests your general knowledge across various topics including geography, science, history, literature, and more!

## Features

- 🧠 **15 Carefully Curated Questions**: A diverse set of general knowledge questions
- 🎯 **Multiple Choice Format**: Choose from 4 options for each question
- 🏆 **Scoring System**: Track your performance with instant feedback
- 📱 **Responsive Design**: Play on desktop, tablet, or mobile
- 🎨 **Modern UI**: Beautiful gradient design with smooth animations
- 🔄 **Replayable**: Play multiple rounds with randomized questions

## How to Play

1. **Start the Game**: Click the "Start Game" button on the welcome screen
2. **Answer Questions**: You'll be presented with 10 random questions from our database
3. **Select Your Answer**: Click on one of the four multiple-choice options
4. **Get Feedback**: See if you're correct and learn the right answer if you're wrong
5. **Continue**: Click "Next Question" to proceed through all 10 questions
6. **View Results**: See your final score and performance message
7. **Play Again**: Start a new game with different questions

## How to Run

Simply open the `index.html` file in your web browser. No additional setup or installation required!

```bash
# Clone the repository
git clone https://github.com/Jake-Lopez/triva-game.git

# Open in browser
open index.html
```

## Topics Covered

The trivia questions span multiple categories:
- 🌍 Geography (capitals, countries, landmarks)

- 📚 Literature (famous authors and works)
- 🏛️ History (major events and dates)
- 🎭 Arts & Culture (famous paintings, mythology)
